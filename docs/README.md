## 提示

贡献请提交 PR 到 debug 分支，感谢。

请保证本地运行 yarn build 后无报错再提交。

## setup hint

推荐使用 vscode 进行编程，推荐使用 vue 拓展\
请先安装 node>=21, yarn>=4.1.1, vue\
目前已有以下插件：\
@vuepress/bundler-vite\
vuepress-plugin-md-enhance\
vuepress\
vuepress-theme-hope
